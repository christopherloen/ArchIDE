ArchIDE

A tiny utility to create, edit and upload your AUR packages.

View on GitHub Download .zip Download .tar.gz

What is ArchIDE?

ArchIDE is a tool to automatically generate, edit and upload AUR packages. Currently ArchIDE is in Pre-alpha version, but I can already list some of its main functionalities:

-clone empty/non-empty repositories from AUR;

-create .install files for your packages;

-use AUR tools like gem2arch to generate, review and upload packages;

-more are coming ...

Further information

ArchIDE is entirely written in Java8. The source code can be found here: https://github.com/christopherloen/ArchIDE.git

Installation instructions

ArchIDE is designed to work on ArchLinux, and of course your computer has to be ready to contribute to AUR. If you have problems while configuring your computer you can follow this wiki: https://wiki.archlinux.org/index.php/Arch_User_Repository#Submitting_packages
